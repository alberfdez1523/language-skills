Install Flow

            
Three steps to get a language skill running locally.

          

          

            

              
1. Clone or download

              

                Publish the repo to GitHub or ship it as a ZIP. The site can
                point users to the repo slug, or they can copy a single skill
                folder manually.
              

            

            

              
2. Install for the agent

              

                Use npx skills add when available, or copy the
                skill into the directory shown in the compatibility matrix. For
                prebuilt layouts, run the bundle script and use
                dist/universal.
              

            

            

              
3. Invoke by skill name

              

                Restart the agent if needed and call the skill explicitly, for
                example $python-engineering or
                $rust-engineering.

Bundle Layout

            
Publish the source repo or generate a universal bundle.

          

          

            

              
Repo distribution

              

                Publish the repository as-is and point users to the install
                configurator. This is the lightest path if you want one canonical
                source of truth.
              

            

            

              
Universal ZIP

              

                Run the included PowerShell script to generate pre-copied
                directories like .codex/skills,
                .claude/skills, and .cursor/skills in
                dist/universal.
              

            

            

              
Manual copy

              

                Every skill is portable. Users can copy a single language folder
                directly into their preferred agent path without touching the
                others.
